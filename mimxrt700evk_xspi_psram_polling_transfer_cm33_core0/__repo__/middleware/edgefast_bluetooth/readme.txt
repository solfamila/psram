NXP Middleware EdgeFast Bluetooth Protocol Abstraction Layer
This repository is for EdgeFast Bluetooth Protocol Abstraction Layer delivery, which is a wrapper layer on top of the bluetooth host stack and based on Zephyr Bluetooth host stack API.
This reposirtory is a part of MCUXpresso SDK overall delivery which is composed of several project deliveries.
Please go to the mcuxsdk to get the MCUXpresso overall delivery to be able to build and run edgefast examples that are based on this reposirtory.
